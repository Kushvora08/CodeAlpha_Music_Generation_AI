How Cadenza works
🎲
Markov chain melody generation
Notes are chosen using weighted step transitions — next notes prefer stepwise motion (step up/down) over large leaps, mimicking how real melodies move.
🌡
Temperature sampling
Low temperature = structured, predictable melodies following scale rules closely. High = more random and experimental choices, producing surprising intervals.
🔊
Web Audio synthesis
Each note plays as a triangle-wave oscillator through a lowpass filter with a custom ADSR envelope — warm attack, natural decay. All done in the browser, zero plugins.
📁
Real MIDI export
Downloads a genuine binary .mid file with proper headers, variable-length delta-time encoding, and note-on/note-off events. Importable into GarageBand, Ableton, FL Studio.
